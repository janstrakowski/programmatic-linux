# Programmatic Linux
Define your Linux system with one Lua script.
## What is it?
Programmatic Linux is basically a Linux system generator - or it was basically the original idea for it to be that but it appears to actually be a folder generator. It takes a file or a directory with Lua code, executes it and (supposedly) generates your desired file hierarchy. You are provided a Lua framework with handy solutions to some well-known Linux distro problems. You are given functions to generate must-have linux components such as bootloader, shell, etc.; you are given a way to define packages with Lua code and build them in containers. You can just copy and paste others people code for the things you don't want to bother yourself because you program in Lua.
